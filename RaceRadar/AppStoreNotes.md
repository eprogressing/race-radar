# App Store Submission Notes - RaceRadar

## App 核心价值 (Core Value Proposition)
RaceRadar 是一款专为大学生打造的学科竞赛信息聚合工具。我们解决了学生在寻找比赛时面临的“信息分散、筛选困难、错过截止日期”等痛点。

**主要功能点：**
1. **结构化筛选**：支持按“奖金金额”、“截止日期”、“竞赛类别”（编程/数学建模/AI数据/创新创业）快速筛选高价值比赛。
2. **智能倒计时**：直观显示剩余天数，提供“即将截止”的醒目提醒，帮助学生合理规划时间。
3. **收藏与管理**：支持本地收藏感兴趣的赛事，方便后续查看。
4. **精美分享**：一键生成包含赛事核心信息（奖金、截止日、二维码）的高清海报，方便在社团或班级群分享。
5. **官方直达**：所有赛事均提供官方源链接（Source URL）及二维码，确保用户能获取最准确的报名信息。

## 审核说明 (Notes for Reviewer)

**1. 关于注册/登录**
本应用**不需要**用户注册或登录即可使用所有功能。我们是一个纯粹的信息聚合工具，不涉及用户账户体系。

**2. 关于数据来源与免责**
- 本应用仅对互联网公开的赛事信息进行结构化展示。
- 应用内不提供直接报名功能，所有报名行为均引导至赛事主办方官方网站（通过 Source URL 跳转）。
- 详情页底部已显著标注“免责声明”和“信息来源”。

**3. 测试路径建议**
为了全面体验 App 功能，建议按照以下路径测试：
1. **首页浏览**：查看赛事列表，尝试向下滑动查看更多。
2. **筛选功能**：点击顶部筛选栏，选择“高奖金 ≥1万”或“即将截止”，观察列表变化。
3. **查看详情**：点击任意卡片进入详情页，查看顶部的 Hero Card 和底部的合规声明。
4. **海报分享**：在详情页点击右上角分享按钮，生成海报，并尝试用另一台设备扫描海报上的二维码（将跳转至官网）。
5. **设置页**：点击底部“设置”标签，查看隐私政策链接和数据状态。
